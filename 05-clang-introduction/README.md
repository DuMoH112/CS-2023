# C LANG

## [Задание 1](./guess_the_number/main.c)

Формулировка задания:
> Задание "Угадать число, загаданное программой"
>
> Программа на языке C, которая загадывает число, и пользователь может угадать его из подсказок программы. 
>
> - "26?" - "Меньше"
>
> - "13?" - "Больше"
>
> - "17?" - "Да!"
